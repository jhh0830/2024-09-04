font-size : 글꼴의 크기를 변경할 때 사용 

        [ 표현법 ]

        선택자 {
            font-size : 크기(px/em/%);
        }

        - px : 픽셀 (고정크기 단위)
        - em : 배수 (가변크기 단위)
                기본 글꼴 크기의 몇배
        - % : 비율 (가변크기 단위)
                기본 글꼴 크기의 몇프로
  • 고정크기 px 테스트
  • 가변크기 em 테스트
  • 가변크기 % 테스트

font-weight : 글꼴의 굵기를 표현할 때 사용 

            [ 표현법 ]

            선택자 {
                font-weight : normal/bold/bolder/lighter/100~900
            }
            - normal : 기본 굵기
            - bold : 굵은 글씨체
            - bolder : 제일 굵은 옵션
            - lighter : 제일 가는 옵션
            - 100~900 : 숫자가 높을 수록 굵어짐
                        (900 == bolder , 100==lighter)
  • 원래 굵기
  • 굵은 글꼴 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

    • font-variant:텍스트 문구를 대문자로 변경할 때 사용 

                [ 표현법 ]

            선택자 {
                font-variant : small-caps;
            }

    font-style : 텍스트 문구를 기울이고자 할 때 사용 

                [ 표현법 ]

            선택자{
                font-style : normal/italic/oblique;
            }

            - normal : 일반 글꼴 (기본값)
            - italic : italic 이라는 기울임 글꼴로 변경하겠다.
            - oblique : 일반 글꼴을 기울이겠다.
    • italic : 기울임 글꼴
    • oblique : 원래 글자를 기울인 셈
